# Recurso estático — Mapa de Resoluciones Técnicas (RT) de la FACPCE

> ⚠️ **RECURSO ESTÁTICO CON FECHA DE CORTE.** Es un **índice de orientación**, no el texto de las RT.
> **Fecha de corte:** `[SIN COMPLETAR — verificar vigencia/versión contra FACPCE antes de citar]`
> **Fuente oficial:** FACPCE (Federación Argentina de Consejos Profesionales de Ciencias Económicas)
> — https://www.facpce.org.ar
>
> 🚨 **Las RT se modifican, derogan, reemplazan y se adoptan/adaptan por cada Consejo Profesional
> (CPCE) provincial.** El **número y objeto** de una RT son relativamente estables (sirven para
> orientarse), pero:
> - La **versión vigente, el texto y la vigencia para una jurisdicción se VERIFICAN contra FACPCE
>   (y el CPCE que corresponda)** antes de citarla como aplicable.
> - **No hay connector** para las RT (FACPCE no expone API; son PDFs). Su cita es **manual + `[verify]`**.
> - Este mapa **orienta** (qué RT mirar para un tema); **no** sustituye leer la RT vigente.

---

## Cómo se usa este recurso

1. Para **orientarse**: ubicar qué RT regula un tema (medición, presentación, ajuste por inflación).
2. Para **citar como aplicable**: recuperar la RT vigente desde FACPCE, confirmar versión y adopción
   por el CPCE de la jurisdicción, y recién ahí citarla. Sin esa verificación → `[verify]`.
3. **No** afirmar un tratamiento contable "según la RT X" sin haber confirmado el texto vigente.

## Recordatorios de grounding

- Este recurso es un **índice de orientación**, no el texto normativo: el tratamiento contable
  concreto se lee de la RT vigente.
- **Marco conceptual estable** (partida doble, qué es un estado contable, rubros) se puede explicar;
  **la cita de "según RT X"** se verifica.
- La adopción de cada RT puede variar por **CPCE provincial**: para Santa Fe, confirmar la adopción
  del CPCE de Santa Fe.
- **Todo estado contable o registración es un borrador para revisión de un contador matriculado.**
